How to Use Clappy

Learn the clap patterns and how the GUI connects audio detection to desktop commands.

Clappy Usage

Clappy listens for real-time microphone input and uses strict clap detection rules to distinguish claps from speech and noise. The algorithm evaluates short-time energy, peak amplitude, and spike strength, then applies a debounce timer to group claps into command patterns.

The default command map is:

  • 1 clap → screenshot saved to the Desktop
  • 2 claps → open a configured URL in the browser
  • 3 claps → lock the screen
  • 4 claps → mute/unmute system audio
  • 5 claps → arm or disarm live actions

In GUI mode, live actions can be enabled even while the application starts in safe mode. The interface shows recorded clap events, a command map, sensitivity controls, and the current noise floor so users can tune detection on the fly.
