Run & Download Clappy

Setup and launch the clap-controlled desktop utility with Python 3.8+.

Run Clappy

Clappy is implemented in a single Python file and requires Python 3.8 or later. The application uses PyAudio for microphone input, PyQt6 for the GUI, NumPy for signal processing, PyAutoGUI for screenshots, pynput for media key handling, and Pillow for image support.

Install the required packages and run one of the following commands from the Clappy folder:

python clappy.py gui
python clappy.py term
python clappy.py check

Use gui to launch the full audio dashboard with live clap detection. Use term for a terminal-only clap listener. Use check to verify dependencies and microphone availability before running.

Clappy can save screenshots to the desktop and supports OneDrive-aware path detection on Windows.
