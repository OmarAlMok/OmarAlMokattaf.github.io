Run HSV Spectrum

Quick instructions to run the PyQt6 HSV decomposition tool.

Requirements

  • Python 3.10+
  • PyQt6
  • NumPy
  • Pillow

Installation

From the project folder, create a virtual environment and install dependencies:

python -m venv venv
venv\Scripts\activate
pip install PyQt6 numpy Pillow

Run

Run the main script (the GUI file) with Python:

python hsv_spectrum.py

The GUI will allow loading an image and exporting the HSV composite.

Notes

The repository includes example images in the `images/` folder used on the subpage for before/after comparisons.
