HSV Spectrum

A PyQt6 tool that decomposes RGB images into hue, saturation, and value channels.

HSV Spectrum Project

HSV Spectrum imports any RGB image and visually decomposes it into three separate channels: hue, saturation, and value. The application converts color pixels to a rich HSV representation, then renders each channel as its own stylized preview so the relationships between color angle, purity, and brightness become clear.

This project was built with PyQt6, NumPy, and Pillow. The user can load an image, decompose it instantly, and export the final HSV composite as a PNG for comparison or further analysis.
