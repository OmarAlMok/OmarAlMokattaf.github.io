# HSV Spectrum

A PyQt6 application that decomposes RGB images into Hue, Saturation, and Value channels.

Requirements
- Python 3.10+
- Install from `requirements.txt`:

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

Run

```bash
# Place the GUI script (e.g. hsv_spectrum.py) in this folder
python hsv_spectrum.py
```

Notes
- Example images are in the `images/` directory used by the project pages.
- If you don't have the GUI script yet, add the provided `hsv` converter script to this folder.
