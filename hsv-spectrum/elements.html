How to Run — HSV Spectrum

Dependencies and quick start.

Dependencies

  • Python 3.10 or newer
  • PyQt6 — GUI toolkit
  • NumPy — numerical arrays
  • Pillow — image I/O

Install

Create and activate a virtual environment, then install required packages:

python -m venv venv
venv\Scripts\activate
pip install --upgrade pip
pip install PyQt6 numpy Pillow

Run

Place the main GUI script (for example `hsv_spectrum.py`) in the project root and run:

python hsv_spectrum.py

Notes & tips

  • If Pillow is missing the required image codecs, install `pillow-simd` or ensure libjpeg/libpng are available.
  • Large images may take several seconds to decompose — try sample images in the `images/` folder first.
  • The Export feature saves a side-by-side composite PNG showing H, S, and V panels.
