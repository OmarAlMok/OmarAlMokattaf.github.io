Online Library Project

The Online Library is a desktop application that lets users manage and borrow books through a PyQt6 GUI connected to a Flask backend. Each book includes metadata such as name, author, publication date, category, and whether it is currently borrowed. All data is stored persistently in a JSON file and reloaded each time the program starts.

The Flask backend provides REST-style endpoints for listing, filtering, searching, adding, deleting, borrowing, and returning books. The GUI communicates with the backend via HTTP requests, allowing users to view the catalog, filter by category or status, search for exact titles, and manage book actions in real time.
