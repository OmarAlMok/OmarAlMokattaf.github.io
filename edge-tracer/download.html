Run Edge Tracer

Quick instructions to run the Canny edge detection demo.

Requirements

  • Python 3.10+
  • PyQt6
  • NumPy
  • Pillow

Installation

Create a virtual environment and install dependencies:

python -m venv venv
venv\Scripts\activate
pip install PyQt6 numpy Pillow

Run

Launch the GUI script with Python:

python edge_tracer_gui.py

The app will process the loaded image and show the edge map and overlay. Use the export button to save side-by-side results.

Notes

Example inputs and outputs are in the `images/` directory and are showcased in the project subpage.
