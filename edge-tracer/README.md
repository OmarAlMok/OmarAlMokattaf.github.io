# Edge Tracer

Auto-calibrated Canny edge detection with a PyQt6 GUI.

Requirements
- Python 3.10+
- Install from `requirements.txt`:

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

Run

```bash
python edge_tracer_gui.py
```

Notes
- The project includes `edge_tracer_gui.py` in this folder.
- Exported results save a side-by-side edge map and overlay PNG.
