How to Run — Edge Tracer

Dependencies and quick start.

Dependencies

  • Python 3.10+
  • PyQt6
  • NumPy
  • Pillow

Install

Create a virtual environment and install packages:

python -m venv venv
venv\Scripts\activate
pip install --upgrade pip
pip install PyQt6 numpy Pillow

Run

Run the main GUI script (for example `edge_tracer_gui.py`):

python edge_tracer_gui.py

Notes

  • Thresholds are auto-calibrated, but very noisy images may require pre-filtering.
  • Exported images appear in the same style shown on the project pages.
